# JavaScript Console Quiz Game

A simple interactive quiz game that runs in the browser console using JavaScript fundamentals.

## Features

- 7 quiz questions with instant feedback
- Case-insensitive answer checking
- Score tracking and percentage display
- Runs entirely in browser console

## How to Run

1. Open browser console (F12 or Ctrl+Shift+J)
2. Copy and paste the code from `quiz.js`
3. Press Enter to start the quiz

## How to Play

1. Answer each question in the prompt
2. Get immediate feedback (correct/incorrect)
3. View your final score at the end

## Technologies Used

- JavaScript (arrays, loops, functions, conditionals)
- Browser Console API (prompt, alert)

## Code Structure

```javascript
quizQuestions[] // Array of question/answer objects
runQuiz()       // Main function containing quiz logic
```

## Learning Outcomes

- Using arrays and objects
- For loops and conditionals
- Functions and scope
- String methods (toLowerCase, trim)
- User input/output handling

## Future Enhancements

- Multiple-choice options
